## Installment


### push
```shell
git add .
git commit -m "message"
git push
```

注意message的规范:“<类别词>: <描述>”

```shell title="message示例"
git commit -m "fix: 修复了xxx的bug"
git commit -m "docs: 更新了xxx的文档"
git commit -m "feat: 增加了xxx功能"
git commit -m "refactor: 重构了xxx"
```
